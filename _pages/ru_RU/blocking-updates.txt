### Обязательно к прочтению

Currently, homebrew is only available for firmware version 3.0.0, but future homebrew and CFW support is planned for all versions up to 5.0.2.

Для обеспечения гарантированного доступа в будущем, необходимо задать для вашей текущей сети адрес DNS сервера (принадлежащего ReSwitched), который блокирует запросы к серверу обновлений Nintendo Switch.

If you are on firmware version 5.0.2 and wish to continue to use online services with updates blocked, you can do so for as long as 5.0.2 is the latest version (as only the latest firmware version is permitted to access online services). Для этого необходимо хотя бы раз подключиться к этим сервисам перед выполнением инструкции.

В дополнение к этому необходимо войти в режим Восстановления / Сервисного обслуживания и удалить все данные обновлений, которые уже могли быть загружены.

### Инструкция

#### Часть I - Конфигурация DNS

Обратите внимание, что адреса этих DNS серверов должны быть настроены для каждой сети, к которой вы подключаете свою консоль, чтобы обновления блокировались!
{: .notice--warning}

1. Откройте Системные настройки
1. Перейдите в `Интернет` -> `Интернет-настройки`
1. Выберите вашу текущую сеть
  + Она обозначена зелёной галочкой рядом с сетью
1. Выберите "Изменить настройки"
1. Измените "Настройки DNS" на "Вручную"
1. Измените "Первичный DNS" на `104.236.106.125`
1. Измените "Вторичный DNS" на `62.210.147.20`
1. Сохраните настройки сети
1. Закройте Системные настройки

#### Часть II - Удаление данных обновления

1. Выключите консоль
1. Включите консоль, зажав кнопку питания + кнопку увеличения громкости + кнопку уменьшения громкости
1. Не выбирайте никаких опций
1. Все загруженные данные обновлений были удалены с консоли при входе в это меню
1. Нажмите кнопку питания для выключения консоли

___

Теперь обновления для вашей консоли заблокированы в данной конфигурации сети.
{: .notice--success}
